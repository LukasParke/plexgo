# Level

An integer log level to write to the PMS log with.
  - 0: Error
  - 1: Warning
  - 2: Info
  - 3: Debug
  - 4: Verbose


## Example Usage

```go
import (
	"github.com/LukeHagar/plexgo/models/operations"
)

value := operations.LevelZero
```
